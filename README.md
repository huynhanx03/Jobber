# Jobber

Jobber is a fast, lightweight bot written in Go that automatically scrapes job portals for matching keywords and forwards opportunities to notification channels.

## Features

### Scrapers
- [x] Indeed
- [x] ITViec
- [x] TopDev
- [ ] VietnamWorks
- [ ] LinkedIn
- [ ] X
- [ ] Facebook
- [ ] Instagram

### Notifiers
- [x] Discord
- [ ] Telegram

## Tech Stack

- **Language:** Go 1.25+
- **Browser Automation:** [go-rod/rod](https://github.com/go-rod/rod)
